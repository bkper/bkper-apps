# Bkper Apps

Open source apps for [Bkper](https://bkper.com).

| App | Description |
|---|---|
| [Exchange Bot](./exchange-bot) | Multi-currency conversion & gain/loss |
| [Inventory Bot](./inventory-bot) | Inventory tracking & COGS (FIFO) |
| [Portfolio Bot](./portfolio-bot) | Investment portfolio tracking |
| [Subledger Bot](./subledger-bot) | Sub-ledger consolidation |
| [Tax Bot](./tax-bot) | Tax calculations |
